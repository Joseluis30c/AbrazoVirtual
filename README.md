# 📖 Descripción

**AbrazoVirtual** es una pequeña aplicación web que permite enviar o generar mensajes simbólicos de apoyo o cariño, simulando un **abrazo virtual**.

El proyecto busca demostrar cómo una aplicación sencilla puede transmitir emociones positivas utilizando tecnologías web básicas.

Este tipo de aplicaciones también es útil como **proyecto educativo**, ya que permite practicar conceptos de:

- Desarrollo web
- Interacción con el usuario
- Lógica básica de programación
- Diseño de interfaces simples
------------------------------------------------------------------------

# 🎯 Objetivo del proyecto

El objetivo de este proyecto es:

- Crear una aplicación simple que permita **enviar o mostrar mensajes positivos**.
- Practicar el **desarrollo web básico.**
- Demostrar cómo pequeñas aplicaciones pueden tener **impacto emocional positivo.**
- Servir como **ejemplo educativo** para principiantes en programación.
------------------------------------------------------------------------

# 🛠 Tecnologías utilizadas

El proyecto utiliza las siguientes tecnologías:

- HTML5 → estructura de la aplicación
- CSS3 → estilos y diseño visual
- JavaScript → lógica e interacción con el usuario
